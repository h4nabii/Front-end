## 标签分类

超文本标记语言(HTML, Hyper Text Markup Language)

HTML标签元素，分成三类

1. 块级元素（block），独占一行

   | tag        | Usage                                           |
   |:-----------|:------------------------------------------------|
   | html       | HTML tag                                        |
   | address    |                                                 |
   | blockquote |                                                 |
   | body       |                                                 |
   | div        | Division                                        |
   | dl         | Description List                                |
   | dt         | Description Term                                |
   | dd         | Description Details                             |
   | fieldset   |                                                 |
   | form       |                                                 |
   | frame      |                                                 |
   | frameset   |                                                 |
   | h1-h6      | Headers or Titles                               |
   | noframes   |                                                 |
   | ol         | Ordered List                                    |
   | ul         | Unordered List                                  |
   | p          | Paragraph                                       |
   | center     |                                                 |
   | hr         | Horizontal Rule                                 |
   | menu       |                                                 |
   | pre        | Pre-formatted                                   |
   | dir        | Directory <font color="red">(Deprecated)</font> |

2. 行级（内联，inline）元素，仅包裹自己,在宽度允许的情况下，只在一行显示

   | Tag     | Usage                                      |
   |:--------|:-------------------------------------------|
   | img     | Image                                      |
   | a       | Anchor                                     |
   | span    |                                            |
   | i       | Italics                                    |
   | cite    |                                            |
   | em      | Emphasize                                  |
   | var     | Variable                                   |
   | address |                                            |
   | small   | Small size & Big size: HTML5 use CSS size. |
   | sub     |                                            |
   | sup     |                                            |

3. 行级块级元素，视下文而定，或者特别指定（行级元素，但拥有块级元素特性）
